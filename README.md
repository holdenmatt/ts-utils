# ts-utils

A few simple zero-dependency Typescript utilities I commonly use across projects.

### Environment checks

- `getEnvironment`
- `isDevelopment`
- `isProduction`
- `isDebug`
- `isBrowser`
- `isServer`

### Basic type guards

- `isString`
- `isFunction`
- `isFile`
- `isArray`
- `isRecord`
- `isError`
- `isNotNull`

### JSON types

- `JSONPrimitive`
- `JSONArray`
- `JSONObject`
- `JSONValue`
